START HERE — HOW TO REUSE THIS CONTEXT

OPTION 1 (recommended):
1. Create a dedicated ChatGPT Project for your LinkedIn profile and career branding.
2. Upload LinkedIn_AI_Career_Context_Master.docx and Profile_and_Project_Update_Forms.docx.
3. Paste New_Chat_Bootstrap_Prompt.txt as the first message.
4. Keep future LinkedIn conversations inside that project and replace the master file whenever information changes.

OPTION 2 (single new chat):
1. Open a new chat.
2. Upload Master_Context.md or LinkedIn_AI_Career_Context_Master.docx.
3. Paste New_Chat_Bootstrap_Prompt.txt.

OPTION 3 (structured import):
Upload Profile_Context.json when you want the assistant to read the information in a structured format.

FILES IN THIS PACK
- LinkedIn_AI_Career_Context_Master.docx — polished human-readable source of truth
- Profile_and_Project_Update_Forms.docx — reusable forms for new projects, roles, recommendations, and profile updates
- Master_Context.md — portable plain-text/Markdown context
- Profile_Context.json — structured profile data
- New_Chat_Bootstrap_Prompt.txt — ready-to-paste instructions for a new chat

IMPORTANT
Review the files before sharing them. Remove confidential model names, client details, internal architecture, or unverified claims. Update the master file whenever your experience or projects change.
